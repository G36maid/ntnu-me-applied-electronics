# 第四章：二極體的應用 (Chapter 4: Diode Applications)

## 課程介紹

### 二極體的主要應用
二極體 (Diodes) 主要用於**直流電源供應器** (DC Power Supply) 或 **AC/DC 轉換器** (AC/DC Converter)，將交流電訊號轉換為直流訊號，為主動電子元件供電，例如：
- 手機/筆記型電腦充電器
- 電視機
- 個人電腦
- 揚聲器

## 基本直流電源供應器

### 四個基本組成部分

一個完整的直流電源供應器可以分為四個部分：

```
┌─────────────────────────────────────────────────────────────────┐
│                    直流電源供應器系統                            │
├─────────────────────────────────────────────────────────────────┤
│  1. 變壓器 (Transformer)                                        │
│     - 調整 AC 訊號至適當的幅度                                    │
│                                                                 │
│  2. 整流器 (Rectifier)                                          │
│     - 將 AC 轉換為脈動 DC 訊號                                   │
│     (半波或全波整流訊號)                                         │
│                                                                 │
│  3. 濾波器 (Filter)                                             │
│     - 消除高頻訊號，平滑 DC 訊號                                 │
│     - 但仍包含漣波 (Ripple)                                     │
│                                                                 │
│  4. 穩壓器 (Regulator)                                          │
│     - 切除漣波，維持恆定 DC 訊號                                 │
│     - 適應負載變化                                               │
└─────────────────────────────────────────────────────────────────┘
```

## 變壓器 (Transformer)

### 基本結構與原理

- **組成**：變壓器具有磁芯 (Magnetic Core) 和兩個繞組 (Windings)
  - 初級繞組 (Primary Winding)
  - 次級繞組 (Secondary Winding)
  
- **工作原理**：透過電磁感應 (Electromagnetic Induction) 傳輸交流電力

- **功能**：
  1. 實現電壓升壓 (Step-up) 或降壓 (Step-down)
  2. 匝數比 (Turns Ratio) n = Ns/Np
     - n < 1：降壓
     - n > 1：升壓

- **變壓器耦合的優點**：
  1. 可以根據需要升壓或降壓
  2. AC 電源與整流器電氣隔絕，避免電擊危險

## 半波整流器 (Half-Wave Rectifier)

### 電路結構

將二極體連接到 AC 電源和負載電阻 RL，形成半波整流器。

### 工作原理

#### 正半週期 (Positive Half-Cycle)
```
輸入電壓 Vin 為正
      ↓
二極體順向偏壓 (Forward-Biased)
      ↓
電流流經負載電阻 RL
```

#### 負半週期 (Negative Half-Cycle)
```
輸入電壓 Vin 為負
      ↓
二極體逆向偏壓 (Reverse-Biased)
      ↓
無電流流經負載 (理想情況)
```

### 工作狀態圖示

```
      順向偏壓                逆向偏壓
    ┌─────┐              ┌─────┐
AC  │     │            AC │     │
    └──┬──┘              └──┬──┘
       │                    │
       ▼                    ▼
    ┌─────┐              ┌─────┐
    │  →  │ 順向        │  ×  │ 逆向
    └──┬──┘ 偏壓        └──┬──┘ 偏壓
       │                    │
       ▼                    ▼
      RL                  RL (無電流)
```

## AC 訊號的參數

### 關鍵參數定義

```
正弦波電壓 v(t) = Vp sin(ωt)

      Vp
       ▲       ┌─────┐
       │      ╱       ╲
       │     ╱         ╲
       │    ╱           ╲
      0├───●─────────────●───→ t
       │   ╲             ╱
       │    ╲           ╱
       │     ╲         ╱
      -Vp    └─────┘
       │
       └───────────────
          Vp-p = 2Vp
```

| 參數 | 符號 | 定義 | 公式 |
|------|------|------|------|
| **峰值電壓** | Vp | 最大電壓值 | Vp = max(v(t)) |
| **峰對峰值電壓** | Vp-p | 峰值到谷值的差 | Vp-p = max(v(t)) - min(v(t)) |
| **平均值電壓** | Vavg | 一週期的平均 | Vavg = (1/T) ∫₀^T v(t) dt |
| **RMS 電壓** | Vrms | 方均根值 | Vrms = √[(1/T) ∫₀^T v²(t) dt] |

### 正弦波的標準值
- **Vavg** = 2Vp/π ≈ 0.637 Vp
- **Vrms** = Vp/√2 ≈ 0.707 Vp

## 半波整流的平均值

### 定義
半波整流輸出電壓的平均值 (Average Value) 是使用直流電壓表測量的值。

### 計算公式

```
Vavg = (1/2π) ∫₀^π Vp sin(ωt) d(ωt)
     = [ -Vp cos(ωt) / (2π) ]₀^π
     = [ -Vp(-1) - (-Vp)(1) ] / (2π)
     = (Vp + Vp) / (2π)
     = Vp/π
```

```
Vavg = Vp/π ≈ 0.318 Vp
```

### 實際測量
- 使用**直流電壓表**測量半波整流輸出時，讀數為 Vavg = Vp/π

## 半波整流的 RMS 值

### 定義
半波整流輸出電壓的 RMS 值 (Root-Mean-Square) 是使用交流電壓表測量的值。

### 計算公式

利用三角恆等式：sin²x = (1 - cos 2x)/2

```
Vrms = √[(1/2π) ∫₀^π Vp² sin²(ωt) d(ωt)]
      = √[(1/2π) ∫₀^π Vp² (1 - cos 2ωt)/2 d(ωt)]
      = √[Vp²/(4π) ∫₀^π (1 - cos 2ωt) d(ωt)]
      = √[Vp²/(4π) [ (ωt) - (sin 2ωt)/2 ]₀^π]
      = √[Vp²/(4π) (π - 0)]
      = √(Vp²/4)
      = Vp/2
```

```
Vrms = Vp/2 = 0.5 Vp
```

### 實際測量
- 使用**交流電壓表**測量半波整流輸出時，讀數為 Vrms = Vp/2

## 位勢障礙的影響 (Effect of Barrier Potential)

### 實用二極體模型

之前我們使用理想二極體模型。為了更實際，現在使用實用二極體模型，考慮 **0.7V 的位勢障礙**。

### 使用建議

| 條件 | 建議模型 | 原因 |
|------|----------|------|
| 峰值電壓 >> 位勢障礙 (≥10V) | 理想模型 | 可忽略位勢障礙的影響 |
| 一般情況 | 實用模型 | 考慮 0.7V 位勢障礙 |

**預設設定：** 除非特別說明，否則使用實用模型（考慮 0.7V 位勢障礙）。

### 輸出電壓計算

考慮位勢障礙後的峰值輸出電壓：
```
Vout(peak) = Vin(peak) - 0.7V
```

## 例題

### 範例 1

**題目：** 繪製整流器的輸出電壓波形，並計算峰值輸出電壓。

給定輸入：Vin(peak) = 100V

**解法：**

```
Vout(peak) = Vin(peak) - 0.7V
           = 100V - 0.7V
           = 99.3V
```

**輸出波形：**

## 峰值逆向電壓 (Peak Inverse Voltage, PIV)

### 定義
峰值逆向電壓 (PIV、逆向峰值電壓) 等於輸入電壓的峰值，二極體必須能夠承受這個重複峰值反向電壓。

### 安全裕度
```
VBR ≥ 1.2 × PIV
```

其中：
- **VBR**：二極體的崩潰電壓 (Breakdown Voltage)
- **PIV**：峰值逆向電壓 = Vin(peak)

二極體的崩潰電壓應至少比 PIV 高 20%。

### 半波整流器的 PIV

在半波整流器中：
```
PIV = Vin(peak)
```

## 練習題

### 練習 2

**題目：** 繪製負載電阻 RL 兩端的輸出電壓波形。如果購買二極體，其崩潰電壓應為多少？

給定：Vin(rms) = 24V

**解法：**

1. 計算峰值輸入電壓：
```
Vin(peak) = Vin(rms) × √2
          = 24V × 1.414
          ≈ 33.94V
```

2. 計算峰值輸出電壓（考慮位勢障礙）：
```
Vout(peak) = Vin(peak) - 0.7V
           = 33.94V - 0.7V
           = 33.24V
```

3. 計算 PIV：
```
PIV = Vin(peak) = 33.94V
```

4. 計算所需崩潰電壓：
```
VBR ≥ 1.2 × PIV = 1.2 × 33.94V ≈ 40.73V
```

**答案：** 應選擇崩潰電壓至少為 **41V** 的二極體。

## 全波整流器 (Full-Wave Rectifier)

### 與半波整流器的比較

| 特性 | 半波整流器 | 全波整流器 |
|------|------------|------------|
| **導電範圍** | 180°（半週期） | 360°（全週期） |
| **效率** | 較低 | 較高 |
| **濾波難度** | 較難 | 較容易 |
| **平均電壓** | Vp/π | 2Vp/π |

### 全波整流的平均值

```
Vavg = (2Vp)/π ≈ 0.637 Vp
```

全波整流的平均電壓是半波整流的 **2 倍**。

## 中心抽頭全波整流器

### 電路結構

使用兩個二極體連接到中心抽頭變壓器的次級繞組。

```
       變壓器次級繞組
    ┌───────────────┐
    │               │
    │  D1      D2   │
    │   │      │    │
────┼───┼──────┼────┼─── 交流輸入
    │   │      │    │
    │   │      │    │
    └───┼──────┼────┘
        │      │
       GND     RL
               │
              GND
```

### 工作原理

#### 正半週期
```
輸入電壓為正
     ↓
二極體 D1 順向偏壓，D2 逆向偏壓
     ↓
電流路徑：變壓器 → D1 → RL → 變壓器中心抽頭
```

#### 負半週期
```
輸入電壓為負
     ↓
二極體 D1 逆向偏壓，D2 順向偏壓
     ↓
電流路徑：變壓器 → D2 → RL → 變壓器中心抽頭
```

### 電流路徑圖示

```
正半週期：
        D1 (ON)     D2 (OFF)
    ─────→──────────────────
           │           │
           ↓           │
           └─── RL ────┘
                │
               GND (中心抽頭)

負半週期：
        D1 (OFF)    D2 (ON)
    ◄────────────────←────
           │           │
           ↓           │
           └─── RL ────┘
                │
               GND (中心抽頭)
```

### PIV

對於中心抽頭全波整流器：
```
PIV = 2 × Vsec(peak) = 2 × Vin(peak)
```

其中 Vsec(peak) 是次級繞組的一半電壓的峰值。

## 練習題

### 練習 3

**題目：** 繪製負載電阻 RL 兩端的電壓波形。二極體的最低 PIV 規格應為多少？

給定：次級繞組 rms 電壓 = 24V

**解法：**

1. 計算次級繞組峰值電壓：
```
Vsec(peak) = 24V × √2 ≈ 33.94V
```

2. 由於中心抽頭，一半繞組的峰值電壓：
```
Vhalf(peak) = Vsec(peak) / 2 ≈ 16.97V
```

3. 計算 PIV：
```
PIV = 2 × Vhalf(peak) = 2 × 16.97V ≈ 33.94V
```

4. 計算所需崩潰電壓：
```
VBR ≥ 1.2 × PIV = 1.2 × 33.94V ≈ 40.73V
```

**答案：** 二極體的最低 PIV 規格應至少為 **34V**，崩潰電壓應至少為 **41V**。

## 橋式全波整流器 (Bridge Full-Wave Rectifier)

### 電路結構

使用四個二極體連接成橋式電路。

```
        D1          D3
    ─────→─────┬────→──────
              │
              ↓
              RL
              │
    ◄────←────┴────←──────
        D4          D2
```

### 工作原理

#### 正半週期
```
輸入電壓為正
     ↓
D1 和 D2 順向偏壓
     ↓
電流路徑：D1 → RL → D2
```

#### 負半週期
```
輸入電壓為負
     ↓
D3 和 D4 順向偏壓
     ↓
電流路徑：D3 → RL → D4
```

對於橋式整流器：

## 電源濾波器 (Power Supply Filters)

### 目的
電源濾波器理想上應消除半波或全波整流器輸出電壓的波動，產生近乎恆定的直流電壓。

### 漣波 (Ripple)
濾波器輸出電壓的小量波動稱為**漣波**。漣波通常是不希望出現的，漣波越小，濾波效果越好。

## 電容器輸入濾波器 (Capacitor-Input Filter)

### 電路結構
將電容器從整流器輸出連接到接地，與負載 RL 並聯。

```
整流器輸出
      │
      ├── C (濾波電容)
      │
      ├── RL (負載)
      │
     GND
```

### 工作原理

#### 第一階段：充電（正半週期的前 1/4 週期）
```
輸入電壓上升
      ↓
二極體順向偏壓
      ↓
電容器充電至峰值電壓
```

#### 第二階段：放電（輸入電壓從峰值下降）
```
輸入電壓低於電容器電壓
      ↓
二極體逆向偏壓
      ↓
電容器只能通過負載 RL 放電
      ↓
放電速率由 RL × C 時間常數決定
```

#### 第三階段：重新充電（下一週期的前 1/4 週期）
```
輸入電壓超過電容器電壓
      ↓
二極體再次順向偏壓
      ↓
電容器重新充電
```

### 工作週期圖示

```
      Vp
       ▲            充電      充電
       │              │        │
       │              │        │
       │        ╱─────╯      ╱───
       │   ╱───╯            ╱
       │  ╱                ╱
       │ ╱                ╱
      0├─────────────────────────→ t
       │                  ╱
       │                 ╱
       │                ╱
       │             ╱──
       │          ╱───
       │       放電
```

## 漣波電壓 (Ripple Voltage)

### 定義
由於充電和放電引起的電容器電壓變化稱為**漣波電壓** (Vr,pp)。

### 時間常數的影響

| 時間常數 | 漣波電壓 | 濾波效果 |
|----------|----------|----------|
| 較小 (RL × C 小) | 較大 | 效果較差 |
| 較大 (RL × C 大) | 較小 | 效果較好 |

### 半波整流 vs 全波整流

| 特性 | 半波整流 | 全波整流 |
|------|----------|----------|
| **輸出頻率** | f (與輸入相同) | 2f (輸入的 2 倍) |
| **濾波難度** | 較難 | 較容易 |
| **峰值間時間** | 較長 | 較短 |

**優點：** 全波整流器的輸出頻率是半波整流器的兩倍，峰值間的時間較短，因此更容易濾波。

## 漣波因數 (Ripple Factor)

### 定義
漣波因數 (r、漣波因數) 是濾波效果的指標。

### 計算公式

```
r = Vr,pp / VDC
```

其中：
- **r**：漣波因數
- **Vr,pp**：漣波電壓（峰對峰值）
- **VDC**：直流輸出電壓（平均值）

### 近似公式

對於半波整流：
```
r ≈ 1 / (2 × f × RL × C)
```

對於全波整流：
```
r ≈ 1 / (4 × f × RL × C)
```

其中：
- **f**：輸入頻率
- **RL**：負載電阻
- **C**：濾波電容

### 降低漣波因數的方法

要降低漣波因數，可以：
1. **增加濾波電容值 (C)**
2. **增加負載電阻 (RL)**
3. **使用全波整流**（頻率加倍）

## 電壓穩壓器 (Voltage Regulators)

### 目的
雖然濾波器可以顯著降低電源供應電壓中的漣波，但 AC 電壓水平仍然不穩定。

**電壓穩壓器**連接到濾波整流器的輸出，維持恆定的輸出電壓（或電流），儘管輸入、負載電流或溫度發生變化。

### 電路結構

```
整流器濾波器輸出
      │
      ├── 穩壓器
      │    │
      │    ├── C2 (輸出電容 0.1~1.0 μF)
      │    │    (改善暫態響應)
      │    ↓
      │   輸出 (恆定直流電壓)
      │
      └── RL
```

### 輸出電容作用
- **範圍**：通常在 0.1 μF 到 1.0 μF 之間
- **功能**：改善暫態響應

### 範例：5V 直流電源供應器

```
    變壓器
       │
       ├── 保險絲 (Fuse)
       │
       ├── 二極體橋 (Diode Bridge)
       │
       ├── 濾波電容 (C1)
       │
       ├── 電壓穩壓器 7805
       │    (輸出 +5V)
       │
       ├── 輸出電容 (C2)
       │
       └── 輸出 (+5V DC)
```

**常用穩壓器編號：**
- **78XX**：輸出 +XX V（正電壓）
- **79XX**：輸出 -XX V（負電壓）

例如：
- **7805**：輸出 +5V
- **7812**：輸出 +12V
- **7905**：輸出 -5V

## 二極體限幅器 (Diode Limiters)

### 定義
二極體限幅器（或稱剪波器 Clippers）用於剪除訊號電壓高於或低於特定水平的部分。

### 正向限幅器範例

```
輸入電壓 Vin
      │
      ├── R1
      │
      ├────┬─── 二極體 (陽極接地)
      │    │
      │    ↓
      │   GND
      │
      └───── Vout
```

### 工作原理

#### 當輸入電壓 > 0.7V
```
二極體順向偏壓
      ↓
點 A 被限制在 +0.7V
      ↓
Vout = 0.7V × [R2 / (R1 + R2)] (電壓分壓器)
```

#### 當輸入電壓 < 0.7V
```
二極體逆向偏壓
      ↓
二極體呈現開路
      ↓
Vout = Vin × [R2 / (R1 + R2)]
```

### 輸出波形

```
輸入：
    Vin
      ▲       ┌─────┐
      │      ╱       ╲
      │     ╱         ╲
      │    ╱           ╲
     0├───●─────────────●───→ t
      │   ╲             ╱
      │    ╲           ╱
      │     ╲         ╱
      │      └─────┘
      │

輸出：
    Vout
      ▲       ┌──────┐
   0.7V       │      │
      │       │      │
      │  ┌────┼────┐ │
      │  │    │    │ │
      │  │    │    │ │
     0├──┼────┼────┼─┼───→ t
      │  │    │    │ │
      │  │    │    │ │
      │  └────┼────┘ │
      │       │      │
```

### 練習題

### 練習 4

**題目：** 繪製 RL 兩端的輸出電壓訊號 Vout。峰值輸出電壓 Vout(p) 是多少？

給定：R1 = 10 kΩ, R2 = 10 kΩ, Vin = ±10V

**解法：**

1. 計算限制電壓（點 A）：
```
VA = 0.7V
```

2. 計算分壓比：
```
k = R2 / (R1 + R2) = 10k / 20k = 0.5
```

3. 計算輸出電壓：

**當 Vin > 0V：**
```
VA = 0.7V (被二極體限制)
Vout = VA × k = 0.7V × 0.5 = 0.35V
```

**當 Vin < 0V：**
```
二極體逆向偏壓（開路）
Vout = Vin × k = Vin × 0.5
```

4. 輸出波形：
```
Vout
     ▲       ┌──────┐
  0.35V      │      │
     │       │      │
     │  ┌────┼────┐ │
     │  │    │    │ │
     │  │    │    │ │
    0├──┼────┼────┼─┼───→ t
     │  │    │    │ │
     │  │    │    │ │
     │  └────┼────┘ │
  -5V       │      │
     ▼       └──────┘
```

**Vout(p) = 5V**

## 二極體箝位器 (Diode Clampers)

### 定義
箝位器將 DC 水平（或偏移量）添加到 AC 電壓。

### 電路結構

```
輸入電壓 Vin
      │
      ├── C
      │
      ├────┬─── 二極體 (陰極接地)
      │    │
      │    ↓
      │   GND
      │
      └───── Vout
```

### 工作原理

#### 第一階段：輸入電壓初始為負
```
輸入電壓為負
      ↓
二極體順向偏壓
      ↓
電容器充電
```

#### 第二階段：負峰值之後
```
電容器充電完成
      ↓
電容器充當另一個 DC 電源
      ↓
輸出電壓被提升一個 DC 水平
```

### RC 時間常數要求
```
RC × 100 > T
```

其中：
- **RC**：電阻電容時間常數
- **T**：輸入訊號的週期

**原因：** RC 時間常數必須足夠大（週期的 100 倍）以維持 DC 水平。否則，電容器可能放電並導致波形失真。

### 輸出波形

```
輸入：
    Vin
      ▲       ┌─────┐
      │      ╱       ╲
      │     ╱         ╲
      │    ╱           ╲
     0├───●─────────────●───→ t
      │   ╲             ╱
      │    ╲           ╱
      │     ╲         ╱
      │      └─────┘
      │

輸出（正箝位器）：
    Vout
      ▲       ┌─────┐
     2Vp      ╱       ╲
      │     ╱         ╲
      │    ╱           ╲
      │   ╱             ╲
      ╱               ╲
     ╱                 ╲
    ╱                   ╲
  0├───────────────────────→ t
```

### 練習題

### 練習 6

**題目：** 假設 RC 足夠大以防止電容器顯著放電。在箝位電路中，預期在 RL 兩觀察到的輸出電壓是多少？

給定：Vin = ±10V

**解法：**

在負半週期，電容器充電至峰值電壓（減去二極體壓降）：
```
VC ≈ Vin(peak) = 10V
```

輸出電壓被提升 VC：
```
Vout = Vin + VC
```

**正半週期：**
```
Vout = +10V + 10V = +20V
```

**負半週期：**
```
Vout = -10V + 10V ≈ 0V (被二極體限制在 -0.7V)
```

**輸出波形：**
```
Vout
     ▲       ┌─────┐
   +20V      ╱       ╲
     │     ╱         ╲
     │    ╱           ╲
     │   ╱             ╲
   0V  ╱               ╲
     ╱                 ╲
    ╱                   ╲
   ╱                     ╲
 0├────────────────────────→ t
```

## 二極體倍壓器 (Diode Multipliers)

### 電壓倍壓器 (Voltage Doubler)

電壓倍壓器是倍數因子為 2 的倍壓器。

### 電路結構

```
次級繞組
      │
      ├── D1 ─┬── C1
      │       │
      │       ├──┬── D2
      │       │  │
      │       │  └── C2
      │       │     │
      │       └─────┤
      │             │
     GND           Vout
```

### 工作原理

#### 正半週期
```
次級電壓為正
     ↓
D1 順向偏壓，D2 逆向偏壓
     ↓
C1 充電至次級電壓的峰值 (Vp)
```

#### 負半週期
```
次級電壓為負
     ↓
D2 順向偏壓，D1 逆向偏壓
     ↓
C1 不能放電
     ↓
C1 的峰值電壓加到次級電壓上
     ↓
C2 充電至約 2Vp
```

### 輸出電壓
```
Vout ≈ 2 × Vp
```

## 摘要

### 重點回顧

1. **直流電源供應器的四個組成部分**
   - 變壓器：調整 AC 電壓幅度
   - 整流器：將 AC 轉換為脈動 DC
   - 濾波器：平滑 DC 訊號，減少漣波
   - 穩壓器：維持恆定輸出電壓

2. **整流器類型**
   - 半波整流器：效率較低，濾波較難
   - 全波整流器：效率較高，濾波較容易
   - 橋式整流器：無需中心抽頭變壓器

3. **AC 訊號參數**
   - Vp：峰值電壓
   - Vp-p：峰對峰值電壓
   - Vavg：平均值電壓
   - Vrms：方均根電壓

4. **半波整流的電壓值**
   - Vavg = Vp/π ≈ 0.318 Vp
   - Vrms = Vp/2 = 0.5 Vp

5. **全波整流的電壓值**
   - Vavg = 2Vp/π ≈ 0.637 Vp
   - Vrms = Vp/√2 ≈ 0.707 Vp

6. **位勢障礙**
   - 實用二極體模型考慮 0.7V 位勢障礙
   - 輸出電壓 = 輸入電壓 - 0.7V

7. **峰值逆向電壓 (PIV)**
   - 半波整流器：PIV = Vin(peak)
   - 中心抽頭全波整流器：PIV = 2 × Vhalf(peak)
   - 橋式整流器：PIV = Vsec(peak)
   - 安全裕度：VBR ≥ 1.2 × PIV

8. **濾波器**
   - 電容器輸入濾波器平滑輸出電壓
   - 漣波因數 r = Vr,pp / VDC
   - 增加 C 或 RL 可降低漣波因數

9. **穩壓器**
   - 維持恆定輸出電壓
   - 輸出電容改善暫態響應
   - 常用編號：78XX（正）、79XX（負）

10. **特殊應用**
    - 限幅器：剪除訊號的特定部分
    - 箝位器：添加 DC 水平到 AC 訊號
    - 倍壓器：提高電壓至輸入的 2 倍

### 公式總結

| 項目 | 公式 |
|------|------|
| **正弦波平均值** | Vavg = 2Vp/π |
| **正弦波 RMS** | Vrms = Vp/√2 |
| **半波整流平均值** | Vavg = Vp/π |
| **半波整流 RMS** | Vrms = Vp/2 |
| **全波整流平均值** | Vavg = 2Vp/π |
| **全波整流 RMS** | Vrms = Vp/√2 |
| **漣波因數（半波）** | r ≈ 1 / (2fRLC) |
| **漣波因數（全波）** | r ≈ 1 / (4fRLC) |
| **PIV 安全裕度** | VBR ≥ 1.2 × PIV |

### 設計考量

| 設計因素 | 考量點 |
|----------|--------|
| **整流器選擇** | 半波（簡單、低成本）vs 全波（效率高、易濾波） |
| **變壓器** | 匝數比、功率額定、隔離需求 |
| **濾波電容** | 電容值、電壓額定、ESR |
| **二極體** | PIV 額定、順向電流、反向恢復時間 |
| **穩壓器** | 輸出電壓、電流容量、散熱 |
