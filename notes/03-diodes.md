# 第三章：二極體 (Chapter 3: Diodes)

## 二極體簡介

### 定義與結構
二極體是由一小塊半導體材料製成的元件，通常是矽，其中一半摻雜成 P 區域，另一半摻雜成 N 區域，中間有 PN 接面和空乏層。

### 組成部分
- **P 區域**：稱為陽極 (Anode)，連接到導電端子
- **N 區域**：稱為陰極 (Cathode)，連接到第二個導電端子
- **PN 接面**：P 區域與 N 區域的交界處
- **空乏層**：PN 接面處無自由載子的區域

## 二極體封裝

### 常見封裝類型

#### 1. 通孔封裝 (Through-hole)
- 傳統的封裝方式
- 適合手動焊接和原型製作
- 引腳較粗，適合插座安裝

#### 2. 表面貼裝封裝 (Surface Mount)
- 現代電子設備的主流封裝
- 適合自動化組裝
- 體積小，適合高密度電路

```
┌─────────────┐           ┌─────────────┐
│  Through-   │           │ Surface     │
│   Hole      │           │  Mount      │
│             │           │             │
│  ──|<──     │           │  ──|<──     │
│   (Axial)   │           │ (SMD)       │
└─────────────┘           └─────────────┘
```

## 順向偏壓 (Forward Bias)

### 定義
順向偏壓是允許電流通過 PN 接面的條件。

### 操作原理
當外部偏壓電壓 V_BIAS 施加到二極體上時：
1. 外部偏壓電壓必須大於位勢障礙電壓 (V_BARRIER)
2. 順向偏壓縮小空乏層
3. 在 PN 接面上產生等於位勢障礙的電壓降

### 位勢障礙電壓值
- **矽 (Silicon)**：約 0.7 V
- **鍺 (Germanium)**：約 0.3 V

### 順向偏壓條件
```
V_BIAS > V_BARRIER
```

### 空乏層變化示意
```
無偏壓狀態：
P 區      N 區
  ───      ───
   +        -
   -        +
  ───      ───
  [空乏層較寬]

順向偏壓：
P 區      N 區
  ───      ───
   +        -
   -        +
  ───      ───
  [空乏層變窄]
```

### 限流電阻
為了保護二極體不被過大電流損壞，必須串聯限流電阻：
```
電源 → 限流電阻 → 二極體 → 接地
```

## 逆向偏壓 (Reverse Bias)

### 定義
逆向偏壓是阻止電流通過二極體的條件。

### 操作原理
當逆向偏壓施加到二極體上時：
1. N 區域的電子被拉向 V_BIAS+
2. V_BIAS- 的電子填滿 P 區域的電洞
3. 空乏層變寬
4. 電流極小（可能會有極微小的洩漏電流）

### 逆向電流特性
- 正常情況下，逆向電流極小（通常在微安級別或更小）
- 可以視為開路狀態

### 空乏層變化示意
```
無偏壓狀態：
P 區      N 區
  ───      ───
   +        -
   -        +
  ───      ───
  [空乏層正常]

逆向偏壓：
P 區      N 區
  ───      ───
   +        -
   -        +
  ───      ───
  [空乏層變寬]
```

## V-I 特性曲線

### 順向偏壓 V-I 特性

#### 小順向電壓階段
當施加順向電壓 V_F 時：
- 如果 V_F < 0.7 V（對矽二極體），順向電流 I_F 非常小
- 二極體呈現高電阻特性

#### 電壓達到位勢障礙
當順向電壓達到 0.7 V 並保持時：
- 順向電流隨偏壓電壓增加而急劇增加
- 二極體呈現低電阻特性

#### 順向特性曲線
```
I_F (電流)
  │
  │      ╱
  │     ╱
  │    ╱      低電阻區域
  │   ╱
  │  ╱
  │ ╱
  │╱
  ├────────────── V_F (電壓)
  │
   高電阻區域
```

### 逆向偏壓 V-I 特性

#### 正常逆向偏壓
當施加逆向偏壓電壓 V_R 時：
- 只有極小的逆向電流 I_R 通過 PN 接面
- 電壓增加時，逆向電流仍然非常小
- 電壓主要降落在二極體上

#### 崩潰電壓 (Breakdown Voltage, V_BR)
當逆向偏壓電壓增加到崩潰電壓 V_BR 時：
- 逆向電流開始急劇增加
- 超過崩潰點後，二極體可能損壞
- 某些特殊二極體（如稽納二極體）設計在此區域工作

#### 逆向特性曲線
```
I_R (電流)
  │
  │
  │
  │         ╲
  │          ╲
  │           ╲
  │            ╲
  │             ╲
  │              ╲
  ├──────────────── V_R (電壓)
  │
   極小電流區域   崩潰區域
       │           V_BR
```

### 完整 V-I 特性曲線

#### 綜合特性
將順向和逆向特性曲線結合，得到完整的二極體 V-I 特性曲線：

```
I (電流)
  │
  │      ╱  順向導通區域
  │     ╱
  │    ╱
  │   ╱
  │  ╱
  │ ╱
  │╱
  ┼──────────────── V (電壓)
  │
  │╲
  │ ╲ 逆向截止區域
  │  ╲
  │   ╲
  │    ╲
  │     ╲
  │      ╲
  │       ╲ 崩潰區域
```

#### 溫度影響
- 在較高溫度下，位勢障礙電壓降低
- 順向電壓降會隨溫度升高而減小
- 逆向電流會隨溫度升高而增加

## 二極體模型

### 模型的目的
為了進行包含二極體的電路分析，需要使用模型來近似二極體的特性。

### 三種模型類型
1. **理想模型** (Ideal Model)
2. **實用模型** (Practical Model)
3. **完整模型** (Complete Model)

## 理想模型 (Ideal Model)

### 特點
- 最不準確的近似
- 使用最簡單的開關模型
- 不考慮位勢障礙和內阻

### 等效電路

#### 順向偏壓
當二極體順向偏壓時，等效於：
```
閉合開關 (導通)
┌────┐
│ ━━│
└────┘
  V_F = 0 V
  R = 0 Ω
```

#### 逆向偏壓
當二極體逆向偏壓時，等效於：
```
開路開關 (截止)
┌────┐
│    │
└────┘
  I = 0 A
  R = ∞ Ω
```

### 範例計算 (理想模型)

#### 順向偏壓
```
電路：V_BIAS = 10 V，R_LIMIT = 1 kΩ，二極體順向偏壓

計算：
V_F = 0 V（理想模型）
I_F = V_BIAS / R_LIMIT = 10 V / 1 kΩ = 10 mA
V_R = V_BIAS - V_F = 10 V - 0 V = 10 V
```

#### 逆向偏壓
```
電路：V_BIAS = -10 V，R_LIMIT = 1 kΩ，二極體逆向偏壓

計算：
V_R = -10 V
I_R = 0 A（理想模型）
V_R (resistor) = 0 V
```

## 實用模型 (Practical Model)

### 特點
- 考慮位勢障礙電壓
- 不考慮動態內阻
- 比理想模型更準確
- 常用於一般電路分析

### 等效電路

#### 順向偏壓
當二極體順向偏壓時，等效於：
```
閉合開關 + 電壓源
┌────┐
│ ━━│──[VF]─┐
└────┘      │
            │
          V_F = V_BARRIER = 0.7 V（矽）
```

#### 逆向偏壓
當二極體逆向偏壓時，等效於：
```
開路開關（與理想模型相同）
┌────┐
│    │
└────┘
  I = 0 A
  R = ∞ Ω
```

### 範例計算 (實用模型)

#### 順向偏壓
```
電路：V_BIAS = 10 V，R_LIMIT = 1 kΩ，二極體順向偏壓

計算：
V_F = 0.7 V（實用模型）
I_F = (V_BIAS - V_F) / R_LIMIT = (10 V - 0.7 V) / 1 kΩ = 9.3 mA
V_R = V_BIAS - V_F = 10 V - 0.7 V = 9.3 V
```

#### 逆向偏壓
```
電路：V_BIAS = -10 V，R_LIMIT = 1 kΩ，二極體逆向偏壓

計算：
V_R = -10 V
I_R = 0 A（實用模型）
V_R (resistor) = 0 V
```

## 完整模型 (Complete Model)

### 特點
- 最準確的近似
- 考慮位勢障礙電壓
- 考慮順向動態電阻 (r'_d)
- 考慮逆向內阻 (r'_R)
- 用於精確的電路分析

### 等效電路參數
- **V_F**：位勢障礙電壓（矽為 0.7 V）
- **r'_d**：順向動態電阻（典型值約 10 Ω）
- **r'_R**：逆向內阻（非常大，約 1 MΩ）

### 等效電路

#### 順向偏壓
當二極體順向偏壓時，等效於：
```
電壓源 + 電阻
┌────┐
│ ━━│──[VF]──[r'_d]─┐
└────┘              │
                    │
  V_F = V_BARRIER + I_F × r'_d
```

#### 逆向偏壓
當二極體逆向偏壓時，等效於：
```
並聯電阻
┌────┐
│    │──[r'_R]─┐
└────┘        │
              │
  I_R = V_R / r'_R
```

### 範例計算 (完整模型)

#### 順向偏壓
```
電路：V_BIAS = 10 V，R_LIMIT = 1 kΩ，二極體順向偏壓
假設：r'_d = 10 Ω

計算：
V_F = 0.7 V（位勢障礙） + I_F × r'_d
I_F = (V_BIAS - 0.7) / (R_LIMIT + r'_d)
    = (10 V - 0.7 V) / (1000 Ω + 10 Ω)
    = 9.21 mA

V_F = 0.7 V + 9.21 mA × 10 Ω
    = 0.7 V + 0.0921 V
    = 0.7921 V

V_R = V_BIAS - V_F = 10 V - 0.7921 V = 9.2079 V
```

#### 逆向偏壓
```
電路：V_BIAS = -10 V，R_LIMIT = 1 kΩ，二極體逆向偏壓
假設：I_R = 1 μA，r'_R = 1 MΩ

計算：
V_R = -10 V
I_R = 1 μA
V_R (resistor) = I_R × R_LIMIT = 1 μA × 1 kΩ = 1 mV
V_R (diode) = V_BIAS - V_R (resistor) = -10 V - 0.001 V = -10.001 V
```

## 三種模型的比較

| 模型類型 | 準確度 | 複雜度 | 順向特性 | 逆向特性 | 應用場景 |
|---------|--------|--------|----------|----------|----------|
| **理想模型** | 低 | 最簡單 | V_F = 0 V, R = 0 Ω | I_R = 0 A, R = ∞ Ω | 初步分析、快速估算 |
| **實用模型** | 中 | 中等 | V_F = 0.7 V, R = 0 Ω | I_R = 0 A, R = ∞ Ω | 一般電路分析、教學 |
| **完整模型** | 高 | 複雜 | V_F = 0.7 V + I×r'_d | I_R = V/r'_R | 精確分析、實際設計 |

## 二極體應用實例

### 1. 整流器 (Rectifier)
將交流電轉換為直流電

```
AC 電源 → 二極體 → 輸出
           (半波整流)
```

### 2. 保護電路
保護敏感元件免受反向電壓損壞

```
電源 → [負載] → 二極體 → 接地
      (逆向偏壓保護)
```

### 3. 電壓調節器
使用稽納二極體穩定電壓

### 4. 信號解調
從載波信號中提取訊息

## 摘要

### 二極體的核心概念

1. **基本結構**
   - P 區域（陽極）和 N 區域（陰極）
   - PN 接面和空乏層

2. **工作模式**
   - **順向偏壓**：V > V_BARRIER，導通
   - **逆向偏壓**：截止，極小洩漏電流
   - **崩潰區域**：V > V_BR，電流急劇增加

3. **關鍵參數**
   - **位勢障礙電壓**：矽 0.7 V，鍺 0.3 V
   - **崩潰電壓**：逆向擊穿電壓
   - **動態電阻**：順向內阻

4. **三種模型**
   - **理想模型**：最簡單，快速估算
   - **實用模型**：平衡準確度和複雜度
   - **完整模型**：最準確，用於精密設計

### 重要公式

```
順向偏壓：
V_BIAS > V_BARRIER
I_F = (V_BIAS - V_F) / R_LIMIT

逆向偏壓：
I_R ≈ 0 A
V_R = V_BIAS

完整模型順向電壓：
V_F = V_BARRIER + I_F × r'_d
```

### 應用注意事項

1. **限流保護**：必須使用限流電阻保護二極體
2. **溫度效應**：溫度升高會降低位勢障礙電壓
3. **逆向電壓限制**：不要超過崩潰電壓（除非是稽納二極體）
4. **封裝選擇**：根據應用選擇適當的封裝類型
